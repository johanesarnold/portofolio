Query Optimization

  • As a Backend Engineer, query result time is very critical. If query result is take a long time to give a response then there is something not optimize with the query. You can optimize your query by doing indexing, select only column you needed, use the correct where like format and avoid using format "%word%". Before running a query for the first time, ensure the results will be desirable and meaningful by using a LIMIT statement.
